Recurring delivery heartbeat. Review every active client project and keep the org moving when no
board prompt is waiting.

Each run:

- Walk every client project: what shipped since last review, what's in progress, what's blocked.
- Confirm the single next useful action per client and that it has an owner.
- Check budget and timeline per client; flag overspend or slippage to the board early.
- Make sure specs are flowing to QA and that nothing is sitting unverified.
- Produce a short board-readable status: per client — shipped / next / blocked / risk.
